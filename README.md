## Project Info
This project is a single-page personal portfolio site built in React. 

## Technology Used
- React
- create-react-app
- SCSS
- Netlify

#### Running the app
- to run locally, fork and clone the repo, run `npm i`, and then run `npm start`
- Otherwise, visit https://www.beccalee.dev/ to use the app
